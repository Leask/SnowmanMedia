What's  Snowman Media?

    Snowman Media  2.0是一个多功能的Windows媒体播放软件。
    
___________
What's New?

    Snowman Media2.0在承继1.0的多种实用功能外，还彻底改进了包
括界面在内的整个软件。Snowman Media2.0重新设计了实用可行的界
面，让使用更轻松。Smm2.0重新规划了功能分类，是新旧功能的使用都
比以往更有条理，实现“傻瓜化”。SmM2.0新增了保存音乐功能，你可
以用它方便的把CD中的音乐保存在电脑里随时欣赏此功能支持音质，声
道及文件大小设置，让你通过设置可以以很小的体积或很好的音质保存
CD的音乐。SmM2.0重新编写了功能更强大的播放列表器，是用它可以更
快的编写出十分个性化的播放列表，以及在列表中继续支持网络播放。
SmM2.0除了在1.0的强大网络功能外还继续加强网络功能，包括直接打
开网络上的Flash文件，直接访问流动网络及其音乐频道以及在线下载
最新版本，升级包及音乐。除上述以外SMM2.0还在1.0基础上作了代码
以及类库的代化，还增加了错误实时处理功能，让其更稳定，可靠。
当然还有其他很多细微的改进，这里就不多说了，大家慢慢感受吧！
   
_____________________
What's the Functions?

    Snowman Media  2.0能播放现在PC和Mac两种硬件架构以及
Windows,Unix及Linus上的共约50种视频，音频文件及图像文件；有
着强大的媒体兼容性；方便简洁美观的操作界面；高度的自动化和实
用性；具有任意大小的拖放窗口和总在最前面功能可以让你一边工作
学习一边享受多姿多彩的多媒体世界；在播放多种视屏时都提供了全
屏视窗和自动鼠标消隐功能是你重拾用电视看电影的感觉；除此以外，
雪人还提供了一个方便实用的播放列表编辑程序；可以用它方便地做
出自己个性化的播放列表。所生成的"*.m3u"文件格式标准和Windows
的Media Player,RealnetWork的RealPlayer及Winamp等各种专业播放
器兼容性极好，可以交互使用；相信大家对Micromedia的flash并不
陌生，使用Snowman Media就可以轻松播放Flash格式的动画及电影；
至于图像现在网上流行的图象格式不下十多种，使用
Snowman Media  2.0就可以不用对这些格式作太大了解它能自动识别
10多种位图和图元文件甚至压缩过的Jpeg和GIF它都能识别并显示出
来。

About Internet!

    在现今国际互联网飞速发展的时代，各种软件部纷纷标榜自己强
大的网络功能。Snowman Media  2.0在设计时也有涉及这方面，他支
持包括Windows的Media player,Apple的QuitTime及RealnetWork的
RealPlayer等十多网络媒体文件实现实时的网络视频和音频的播放；
另外Snowman Media  2.0还支持网络升级有兴趣的朋友可以到
"www.h2ont.com"上看看或者给我们发E-mail。
    
____________________
What's Insufficient?

    上述的是Snowman Media  2.0的多媒体功能，相信大家看了对它
也有一定了解了吧。因为本人水平及开发资源有限，它还存在这样那
样的很多不足，如：它没有Media player的媒体库等方便的收藏功
能，没有超级解霸和东方影都优秀的VCD支持功能，没有Realplayer，
QuitTime的强大网络功能；没有Winamp的占用资源及“体积”小的优
势。也没有它们专业稳定性和可换肤的漂亮界面，和各种各样的个性
化播放器比较真有不少逊色。希望在大家支持和使用的同时给我反馈
一些忠实的意见和建议让我可以把Snowman Media做的更好，更出色。
我们相信Snowman Media的"3.0"会更加优秀的，因为有着你们的支
持。

______________
Why I Made It?

    为什么会做Snowman Media  2.0？
    Snowman Media  1.0已经发布差不多两个月那个时候我们就想过
要开发"2.0"但由于要应付地理会考我们一直没有时间直到上周四(6.
7.2001)接到有关全国计算机设计比赛的通知我们流动网络才决定要马
上开发。于是经过一周多的努力Snowman Media  2.0终于和大家见面
了。在一周的时间内既要重新规划界面又要编写代码时间很紧所以一
定有不少不是很如意的缺陷希望大家向"1.0"那样多多批评指正。
