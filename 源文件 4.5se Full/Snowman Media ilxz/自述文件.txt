What's Snowman Media 4se?

    Snowman Media 系列是一个高度集成的媒体播放及管理软件；
Snowman Media 4se（下简作 Snowman）是 Snowman Media 系列
的最新版本。4se 为 Snowman Media 4.x 的版本版本名。
    Snowman 能播放 Windows、Unix、Mac 及 Linux 上的共约 50
种视频，音频文件及图像文件；有着强大的媒体兼容性；简洁、个
性的操作界面。经过 4 个主版本的发展后并不单单只是一个媒体播
放器,更是你在本地计算机和 Internet 上播放和管理多媒体的得力
助手。
    Snowman 为您提供前所未有的多媒体体验：你除了可以轻松播
放绝大多数类型的媒体外，Swnoamn 提供更精彩的家庭多媒体扩展
方案。使用它你可以轻松捕获来自 CD、VCD、DVD 上的精彩内容，
并方便发布到互联网与他人共享；可以连接扫描仪、数码相机、数
字摄像机以及其他兼容媒体设备以交换媒体，还可以在个人电脑与
MD 机、MP3 播放机、MP3 手机和 WMA 播放机等数数码随身听之间
交流音乐。
    Snowman 具备强大的网络功能，网络播放兼容 Microsoft
Windows Media Player、Apple QuitTime、Macromedia Flash 及
RealNetWorks RealPlayer 所有的媒体格式。方便的调谐收音机让
你轻松收听来自全球数千个网络广播电台的广播节目并可以通过自
动升级更新新增的电台列表。在你播放媒体或电台节目时你可以通
过媒体信息服务搜索关于当前媒体的艺术家、唱片集等信息。实现
交换和共享媒体内容，通过互联网、和无线网络把你珍贵的媒体作
品传送到你心爱的人的计算机上，例如是你亲手做的动画贺卡。
   总之使用 Snowman Media 4se 你可以轻松体验个性、轻松和愉
悦的数字多媒体生活。

What's New?

    改进外观：加入快速任务功能，使操作更便捷；
    改进同步媒体列表：加入顺序调整，和歌曲标题、艺术家信息
的显示；
    改进书签和断点续播：实现书签菜单中显示歌曲标题、艺术
家、停止时刻，使续播功能支持 CD、DVD 光盘；
    媒体格式方面：新版本加入了对 DVD、RealMedia 和 Mpeg4
编码媒体的支持，当然还包括精彩的 Flash 矢量影片；
    对光盘的支持也有所突破：除了支持 CD、VCD 和 DVD 之外还
智能的支持各种非正统的媒体光盘如 Mp3 和各种演示媒体光盘，
实现智能搜索；
    CD 音乐：新增 Snowman Media 4se CD-Casket 插件，实现复
制 CD 音乐到个人计算机；
    针对 VCD 的改进：实现 VCD 双声道选择；
    针对网络：采用新的播放内核，不仅解码速度更快而且播放质
量也大大提高；
    新增的收音机调谐器：轻松收听全世界的网络电台；
    加入 HDTV 插件：轻松播放 HDTV 高清晰度数字电视，HDTV 高
清晰度数字电视提供水平 500 分辨率线的 DVD 质量高清晰度视
频。
    针对 Real Media 的改进：嵌入 Real Media 播放组件；
    新增媒体信息服务：获取和更改媒体信息标签。
    以上是较大的改进，其他不少细微改进有待大家慢慢发掘。

Why I Made It?

    为什么会做 Snowman Media 4se？
    Snowman Media 3.0 已经发布很久了，针对存在较多的缺点许
多朋友和网友都希望我能尽早开发更新版本。但因为要应付高
二学科会考所以一直没有足够的时间和精力搞开发。终于迎来了暑
假，在这一个多月的奋斗后，ilxz 的测试版本终于在 8 月初面
世，上传到太平洋电脑网和华军软件园。8 月 20 分析发布测试版
之后收到的宝贵意见并开始修改。经过修改和第二个测试版的试用
最终版终于逾期于 9 月 13 日发布了。本版本 2003 年 10 月 16
日修改完善的参赛特别版 4se，修改了大量的 Bug 和对算法做了
较大的优化。

Snowman 做的更好，更出色。我肯定 Snowman 的 5.0 会更加优

In the Future.

    “每一个人都要有一个梦，要不日子就会很难过下去”这是
《想飞 Princess-D》上的一句话。我有同感，我的 Snowman Media
就是我的半个梦。Snowman Media 是我的心血，作为一个软件的作
者当然希望自己的作品能不断进步、成长我也不例外。现阶段我没
有能力把 Snowman Media 做的像大公司的播放器那样出色，不是我
不想，而是自己水平有限。当然我说的是现在，而在未来，我对自
己有信心。
    短期内我不会继续搞开发了，原来的英文版、各种功能插件和
Linux 下的 Snowman Media 这些计划又要压后了。没办法，我已经
快高考了要应付紧张的学习，我已也不能再那么任性整天对这电脑
了。Snowman Media 的下一个版本我会在高考后编写，希望你们会
期待！
