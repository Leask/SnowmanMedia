How to use
  SnowmanMedia的主界面是一个CD播放器
  单击雪人的扫帚，能打开控制面板。
  单击控制面板的选项卡,能选择你想要的功能。
  单击雪人身上的纽扣，能切换CD播放时间的显示方式。（已播放时间／剩余时间／唱片剩余时间）
  单击中间的喇叭能打开媒体文件
  选择CD曲目的下拉菜单能跳转到你想要的CD曲目
  单击音量控制按钮调整到合适的音量
保存音乐:

在控制面板单击MusicBox选项卡,选择Save Music打开保存音乐窗口

  确保光驱中有CD唱片，单击“刷新CD信息”。在左边的框中双击要保存的曲目，若要取消选择，就在中间的框中双击要取消的曲目。选择好保存的质量（质量愈好，保存成的文件愈大），还有保存的路径，单击“开始”。当一首曲目播放完后，保存工作也就完成。
播放列表编辑器：

在控制面板单击Play List选项卡，选择Edit list打开播放列表编辑器

  在最边的框中选择磁盘目录，在文件类型下拉菜单中选择要加入播放列表的媒体类型，单击“过滤”按钮，在中间的框中单击曲目名称加入右边的框，若要取消选择，就单击“清除”按钮。完成后单击“生成播放列表”按钮，弹出保存文件对话框，选择好保存路径和文件命后按“保存”就完成了。
