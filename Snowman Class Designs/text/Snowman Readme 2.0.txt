新特性：
   Snowman Media2.0在承继1.0的多种实用功能外，还彻底改进了包括界面的整个软件。Snowman Media2.0重新设计了实用可行的界面，让使用更轻松。Smm2.0重新规划了功能分类，是新旧功能的使用都比以往更有条理，实现“傻瓜化”。SmM2.0新增了保存音乐功能，你可以用它方便的把CD中的音乐保存在电脑里随时欣赏此功能支持音质，声道及文件大小设置，让你通过设置可以以很小的体积或很好的音质保存CD的音乐。SmM2.0重新编写了功能更强大的播放列表器，是用它可以更快的编写出十分个性化的播放列表，以及在列表中继续支持网络播放。SmM2.0除了在1.0的强大网络功能外还继续加强网络功能，包括直接打开网络上的Flash文件，直接访问流动网络及其音乐频道以及在线下载最新版本，升级包及音乐。除上述以外SMM2.0还在1.0基础上作了代码以及类库的代化，还增加了错误实时处理功能，让其更稳定，可靠。当然还有其他很多细微的改进，这里就不多说了，大家慢慢感受吧！
