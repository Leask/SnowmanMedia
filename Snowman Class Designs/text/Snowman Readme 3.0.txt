What's  Snowman Media?

    Snowman Media 系列是一个多功能的 Windows 媒体播放及管理
软件。
    

___________
What's New?

    Snowman Media  3.0 在承继 2.0 的多种实用功能外，还彻底改
进了包括界面在内的整个软件。Snowman Media  3.0 重新设计了实
用可行的界面并加入外观定制功能。SmM3.0 重新规划了功能分类，
使新旧功能的使用都比以往更有条理，实现“傻瓜化”。SmM3.0 新
增了同步播放列表播放模式，你可以用它方便地批量播放媒体文件，
让你通过设置随时定制同步列表。SmM3.0 还加入了断点续播、媒体
书签、定时关机等人性化功能，服务你的媒体生活细微体贴。在媒体
格式方面SmM3.0 加入了对 DVD(Mpeg4) 、RM 和 FLX 的支持让你可
以用一个软件播放几乎所有的媒体格式。SmM3.0 对光盘的支持也有
所突破，除了支持 DVD 和 VCD 之外还智能的支持各种非正统的媒体
光盘如 Mp3 和各种演示媒体光盘。SmM3.0 重新编写了功能更强大的
播放列表器，是用它可以更快的编写出十分个性化的播放列表，以及
在列表中继续支持网络播放。SmM3.0 除了在 2.0 的强大网络功能外
还继续加强网络功能，包括强大的网络向导让你轻松连接 Internet
并加入动态带宽调整技术让你即可得到最好的播放质量同时不影响其
他程序访问网络。除上述以外 SmM3.0 还在 2.0 基础上作了代码以
及类库的代化，还增加了错误实时处理、智能升级和修复等功能，让
其更稳定，可靠。当然还有其他很多细微的改进如。这里就不多说
了，大家慢慢感受吧！
 
  
_____________________
What's the Functions?

    Snowman Media  3.0 能播放现在 PC 和 Mac 两种硬件架构以及
Windows、Unix 及 Linus 上的共约 50 种视频，音频文件及图像文
件；有着强大的媒体兼容性；方便简洁、美观、个性的操作界面。
Snowman Media 系列经过 3 个主版本的发展后并不单单只是一个媒
体播放器,更是你在计算机和 Internet 上播放和管理多媒体的得力
助手。Snowman Media 为您提供前所未有的多媒体体验。使用它你
可以轻松播放绝大多数类型的媒体文件、可以在线播放世界范围内互
联网上提供的电影、以及创建您本地计算机及网络上所有媒体的自定
义列表。除此以外还包括先进的断点续播和媒体书签支持、个性化的
自定义界面、各项人性化的自动操作等等。其次 Sm.M. 并不仅限于
用来欣赏媒体，借助新增的可升级性能你可以下载如绘图、作曲辅助
等组件来创作你自己的媒体.让你觉得数字多媒体生活原来可以如此
个性、轻松和愉悦。

About Internet!

    在现今国际互联网飞速发展的时代，各种软件部纷纷标榜自己强
大的网络功能。Snowman Media  3.0 在设计时也有涉及这方面，他
支持包括 Windows 的 Media Player、Apple 的 QuitTime、
Macromedia 的 Flash 及 RealnetWork 的 RealPlayer 等十多网络
媒体文件实现实时的网络视频和音频的播放；另外 Snowman Media
3.0 还支持网络升级有兴趣的朋友可以到 www.h2ont.com 上看看或
者给我们发 E-mail。

Why I Made It?

    为什么会做 Snowman Media  3.0？
    Snowman Media  2.0 已经发布很久了，那个时候我要应付全国
比赛因为时间和精力问题我没有把我想要做的都实现，就想过要开发
3.0 但由于要应付学习我把计划推迟到放暑假。放假后经过我两个月
忘我的疯狂努力和几个热心好朋友的支持帮助下Snowman Media  3.0
终于和大家见面了。

In the Future.

    Snowman Media 的未来？我是个不现实的人，这个问题我答不
上。Sm.M. 是我的心血，作为一个软件的作者当然希望自己的作品能
不断进步、成长我也不例外。我不能把 Snowman Media 做的像大公
司的播放器那样出色，不是我不想而是自己水平有限。当然我说的是
现在，而在未来我对 SmM 有信心。因为我相信我会像小朱说那样不
断进步，加上我身边还有那么多好朋友的支持和鼓励。
    短期内我会继续完善 SmM3.0 推出英文版以及推出一系列的功能
插件供用户升级。另一方面我会学习 Linux 下的编程语言发展
Linux 下的 Snowman Media。可实际一点我还是在校学生还要应付学
习可没有那么多时间让我“工作”，那就只能一步一步慢慢来了。
